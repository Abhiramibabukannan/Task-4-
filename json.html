}); Read about the diffrence between windows ,Screen & Document in javascript Window Object - The window object represents the browser window or frame. - It's the global object in JavaScript, and all other objects, functions, and variables are its properties. - Properties: - window.location: The current URL. - window.history: The browser's history. - window.navigator: Information about the browser. - window.screen: The user's screen. - Methods: - window.alert(): Displays an alert box. - window.confirm(): Displays a confirmation box. - window.prompt(): Displays a prompt box. - window.open(): Opens a new window or tab. - window.close(): Closes the current window. Screen Object - The screen object represents the user's screen. - It's a property of the window object. - Properties: - screen.width and screen.height: The screen's resolution. - screen.availWidth and screen.availHeight: The available screen space. - screen.colorDepth: The screen's color depth. - screen.orientation: The screen's orientation. - Methods: None Document Object - The document object represents the HTML document. - It's a property of the window object. - Properties: - document.title: The document's title. - document.URL: The document's URL. - document.referrer: The referrer URL. - document.body: The HTML body element. - document.head: The HTML head element. - Methods: - document.querySelector(): Selects an element. - document.createElement(): Creates a new element. - document.addEventListener(): Attaches an event listener. - document.write(): Writes HTML content. Key Differences - Scope: window is the global scope, while document is a property of window. - Purpose: window handles browser-related tasks, document handles HTML document-related tasks, and screen provides screen-related information. - Accessibility: window and document are accessible globally, while screen is accessible via window.screen.
